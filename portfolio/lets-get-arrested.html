Let's Get Arrested!

Project Details

  • Category : Personal
  • Link : Click me

In Japan, a schoolgirl was caught by the police for putting a link to a website with an endless loop of alert in JavaScript.

A movement of protestation grew among developers around the world : #LetsGetArrested4JSCode. Kimikazu Kato created a GitHub repo with the JS code to be easily published using GitHub Pages.


If you want to learn more about this story in Japanese, French or English.
